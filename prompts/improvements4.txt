₿itTrade UI Improvements & Enhancements
========================================

## Design System Improvements

### 1. Monochromatic Dark Theme
- Consistent dark color palette throughout the app
- Primary background: #0f0f0f (very dark gray)
- Secondary background: #1a1a1a (dark gray)
- Card backgrounds: #262626 (medium dark gray)
- Borders: #404040 (lighter gray)
- Text: #ffffff (white) for primary text
- Muted text: #a3a3a3 (light gray)

### 2. Icon System Standardization
- Exclusively use lucide-react icons throughout the app
- All icons should be white (#ffffff) for consistency
- Standard icon size: 20px for most UI elements
- Larger icons (24px) for primary actions and navigation
- Remove any non-lucide-react icons or emoji

### 3. Bitcoin Symbol Consistency
- Use ₿ symbol consistently for Bitcoin throughout the app
- Replace all instances of "BTC" text with ₿ symbol
- Maintain proper formatting: ₿0.00012345 (8 decimal places)
- Use formatBitcoin utility function for consistent formatting

## Component Improvements

### 4. Trading Modal Enhancements
- Add percentage quick-select buttons (25%, 50%, 75%, 100%)
- Real-time calculation display as user types
- Better visual feedback for buy/sell actions
- Improved spacing and button layouts
- Clear action buttons with proper hover states

### 5. Transaction Cards (History Page)
- Improved spacing between elements
- Better visual hierarchy with proper text sizing
- Consistent icon usage (lucide-react only)
- User-friendly transaction type labels:

  - "INR Deposit" for DEPOSIT_INR
  - "₿ Purchase" for BUY
  - "₿ Sale" for SELL
  - "INR Withdrawal" for WITHDRAW_INR

- Proper amount formatting with ₿ symbol

### 6. Dashboard Cards (Home Page)
- Clean, minimal card design
- Consistent spacing and padding
- Proper visual hierarchy
- White lucide-react icons for all elements
- Balance cards with clear typography
